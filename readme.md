Installation instructions

Set the appropriate database constants in .env (rename the .env.example)

DB_CONNECTION=
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

Once the database configuration is set just run the setup script:

php setup.php
